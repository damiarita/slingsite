## 2. Description du service
Slingsite fournit un outil de compression d'images et de vidéos côté client qui traite les fichiers entièrement dans le navigateur web de l'utilisateur. Le Service :
- Génère plusieurs versions compressées des fichiers téléchargés, optimisées pour différents appareils et formats
- Fonctionne entièrement côté client, sans traitement ni stockage côté serveur
- est fourni gratuitement
